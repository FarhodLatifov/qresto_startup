# 🚀 Быстрый старт - Исправление отображения заказов

## Проблема решена! ✅

Теперь админ панель будет показывать реальные заказы из базы данных.

## Что нужно сделать СЕЙЧАС:

### Шаг 1: Применить миграции к базе данных

**Вариант A - Через SQL Editor (5 минут):**

1. Откройте https://app.supabase.com
2. Выберите ваш проект
3. Кликните **SQL Editor** в левом меню
4. Кликните **New query**
5. Откройте файл `supabase/migrations/00001_initial_schema.sql` 
6. Скопируйте ВСЁ содержимое файла
7. Вставьте в SQL Editor и нажмите **Run**

**Вариант B - Автоматически:**

1. Добавьте в `.env`:
   ```env
   VITE_SUPABASE_SERVICE_ROLE_KEY="ваш_ключ"
   ```
   (ключ берем в Settings → API)

2. Запустите:
   ```bash
   npx tsx apply-migrations.ts
   ```

### Шаг 2: Проверить работу

1. Запустите dev сервер (если не запущен):
   ```bash
   npm run dev
   ```

2. Откройте гостевое меню: http://localhost:3000/menu/demo-rest/table/1

3. Добавьте что-нибудь в корзину и оформите заказ

4. Перейдите в админ панель: http://localhost:3000/admin/dashboard

5. **Заказ должен отобразиться!** 🎉
